# Features

- Rxjava & RxAndroid
- HTTP Requests & Retrofit
- Export Excel File
- Fingerprint Login
- Facebook Login
- Google Login
- MpAndroidCart
- Push Notifications
- Share: Facebook , Instagram , Messenger , Whatsapp
- Swipe To Delete
- hCaptcha
- Pay by Braintree service
- Mapbox Maps
- AES encryption
- Hawk
- ProGuard
- Privacy Policy & Terms of use
- Agora SDK
- Requests list
- Projects list
- Profile screen
